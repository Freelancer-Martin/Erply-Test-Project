Introduction

Requirements

You will need the following List to make it work

  1. Create database "api_db" with MySQL, don't worry about tables, tables are created automatically
  2. Use different endpoints to create, delete, update and search data
  3. All data is shown has JSON via HTTP, sake of this example everything happens via URL request)

Be careful while editing the files. If not edited properly, the design layout may break completely.
No support is provided for faulty customization.

Create Product #back to top

How to Create Product Endpoint: http://localhost/API/product/create.php

Parameter Parameter Example
name Eve
price 300 

                                  Json payload example {"name":"sucky","price":250}
                                  Full URL: http://localhost/API/product/create.php?{"name":"sucky","price":250}
